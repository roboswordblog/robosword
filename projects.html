Free Typing
This is a typing clone that was made in python using the pygame library. It uses a bunch of random words from a list. Using pygame, the y is increased when it hits the boundry of the screen. It checks every second which keys are being typed, and then it makes a rectangle with the color if you typed it. It tracks wpm and accuracy. Then when you go to the end screen, you can hit the replay button and replay all the words which you got correct or wrong.
